개인정보처리방침

TimeFit은 모임 일정과 장소를 조율하는 데 필요한 최소한의 정보만 처리합니다.

처리하는 정보

모임 제목·설명·주최자 표시 이름, 참여자 표시 이름, 가능한 날짜와 시간, 사용자가 선택한 출발 위치, 모임 설정과 최종 확정 정보를 저장합니다. 회원가입이나 주민등록번호, 결제 정보는 요구하지 않습니다.

이용 목적

입력 정보는 가능한 시간의 교집합 계산, 이동 균형 지점 추천, 참여 현황 표시, 모임 관리와 결과 공유를 위해 사용합니다.

공개 범위

초대 링크와 결과 링크를 알고 있는 사람은 모임 정보와 참여자 표시 이름, 가능한 시간을 볼 수 있습니다. 공개 결과에서는 출발 주소를 숨기고 위치 좌표의 정밀도를 낮춥니다. 주최자 관리 화면과 CSV에는 참여자가 입력한 위치 정보가 표시될 수 있습니다.

브라우저 저장

같은 기기에서 응답을 수정하거나 최근 모임을 다시 관리할 수 있도록 응답 수정 토큰과 주최자 관리 토큰을 브라우저의 로컬 저장소에 보관합니다. 브라우저 데이터를 삭제하면 해당 권한을 다시 찾지 못할 수 있습니다.

외부 서비스

데이터 저장과 서버 기능에는 Supabase를, 지도·주소·장소 검색에는 Kakao Maps를 사용합니다. 각 서비스의 처리 방침이 함께 적용될 수 있습니다.

보관과 삭제

모임 정보와 참여 응답은 주최자가 모임을 삭제할 때 함께 삭제됩니다. 참여자는 같은 기기에 저장된 수정 권한으로 자신의 응답을 삭제할 수 있습니다.

안전한 이용

정확한 집 주소 대신 가까운 역이나 동네 중심을 입력하는 것을 권장합니다. 주최자 관리 링크에는 관리 권한이 포함되므로 다른 사람에게 전달하지 마세요.
